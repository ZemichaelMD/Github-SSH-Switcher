# Git-SSH-Switcher
SSH account switcher tool allows the creation of multiple ssh "accounts" for github. 

## Installation
---
```shell
./setup.sh
```
*Please don't use sudo for the setup. It will fail*


<br>

## Usage
---
### LIST ACCOUNTS
```shell
$ ssh-list
```

### SWITCH ACCOUNT
```shell
$ ssh-switch
```

### CREATE
```shell
$ ssh-create
```

### REMOVE
```shell
$ ssh-remove
```

### CLEAR
```shell
$ ssh-clear
```
